Recently solved an interesting leetcode question that I wanted to share:

**137. Single Number II**

*Given an integer array nums where every element appears three times except for one, which appears exactly once. Find the single element and return it.
You must implement a solution with a linear runtime complexity and use only constant extra space.*

The intuition behind the solution is to sum the value at each bit of every `num` in the input then mod 3. For example, let the input be `[2,2,2,3]`, its binary representation is `[10, 10, 10, 11]`, sum at each bit gives `41`, then mod 3 gives `11` (which is the binary of the answer `3`).

To implement concretely, we use two integers `a` and `b` (initialized to 0) to store the intermediate sums (mod 3). For bit position `i`:
| `b[i]`| `a[i]`|sum at i(mod 3)|
| :-: | :-: | :-: |
| 0   | 0   | 0   |
| 0   | 1   | 1   |
| 1   | 0   | 2   |

The interesting bit ;) is to make sure the sum (`a` and `b`) gets updated correctly for each `num`. We will use Karnaugh maps to find the correct updates.

Reading off the conjunctive normal form, we find the updates should be:
```
new_a = ~b & (a | num) & (~a | ~num)
new_b = (a | b) & (~b | ~num) & (~a | num)
```

The solution in Python is thus:
```
def singleNumber(self, nums: List[int]) -> int:
    a, b = 0, 0
    for num in nums:
        a_ = ~b & (a | num) & (~a | ~num)
        b = (a | b) & (~b | ~num) & (~a | num)
        a = a_
    return a | b
```
